Logo Guidelines

The Amply wordmark and how to use it

Amply

The Wordmark

The Amply logo is a wordmark—the name "Amply" set in Lexend Black (900). This approach keeps our identity simple, recognizable, and accessible.

The wordmark should never be recreated or approximated. Always use the official wordmark files or render it using Lexend Black at the correct specifications.

Clear Space

The wordmark needs room to breathe. Maintain a minimum clear space around the logo equal to the height of the letter "A" in the wordmark.

This ensures the logo remains legible and uncluttered in any context.

Amply
Minimum clear space = 1× height of "A"

Minimum Size

To maintain legibility, the wordmark should never appear smaller than the minimum sizes shown below.

Amply 64px / 48pt
Amply 40px / 30pt
Amply Minimum: 20px / 15pt

For digital applications, 20px is the absolute minimum. For print, 15pt (approximately 5mm in height) is the minimum.

Accessibility

Amply Teal (#05668D) on white has a contrast ratio of 6.4:1, exceeding WCAG AA requirements (4.5:1) for normal text and AAA for large text. WCAG AA

White text on Amply Teal background also achieves 6.4:1 contrast.

Implementation

CSS

/* Brand colors */ --amply-primary: #05668D; --amply-dark: #044a66; --amply-light: #e8f4f8; /* Wordmark */ .amply-wordmark { font-family: 'Lexend', sans-serif; font-weight: 900; letter-spacing: -0.02em; color: var(--amply-primary); }
