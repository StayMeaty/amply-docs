# Platform Tools
*Everything You Need to Manage Donations*

Amply provides a comprehensive suite of tools for collecting donations, managing donors, and demonstrating impact. This document covers what's available.

## Donation Collection

### White-Label Donation Pages

Fully branded donation pages hosted on Amply:

- **Custom URL**: yourorg.amply-impact.org or your own domain
- **Your branding**: Logo, colors, fonts
- **Flexible amounts**: Preset options or custom amounts
- **Recurring giving**: Monthly, quarterly, annual options
- **Multiple currencies**: Accept donations in various currencies
- **Mobile-optimized**: Works on all devices

### Embedded Widgets

Add donation functionality to your own website:

- **Donation buttons**: Simple "Donate" buttons that open Amply checkout
- **Inline forms**: Embed full donation forms on any page
- **Progress bars**: Show fundraising progress
- **Donor walls**: Display recent supporters (with consent)

→ [Widgets documentation](../integrations/widgets.md)

### Campaign Pages

Create focused campaigns for specific projects:

- **Dedicated URLs**: Share campaign-specific links
- **Goals and progress**: Set targets, show progress
- **Time limits**: Create urgency with deadlines
- **Updates**: Keep supporters informed
- **Fundraiser enablement**: Let supporters create their own pages

## Donor Management

### Donor Profiles

Comprehensive view of each supporter:

- Giving history across all campaigns
- Communication preferences
- Notes and tags
- Contact information

### Communication Tools

Stay connected with supporters:

- **Tax-deductible receipts**: Generate compliant tax receipts for donors
  - Australia DGR-compliant template available
  - Your branding: logo, letterhead, acknowledgment message
  - Automated: real-time after donation, or batch/annual
  - Verifiable: links to public ledger entry
  - *Additional jurisdictions coming: UK, Canada, Germany*
- **Thank-you messages**: Customizable acknowledgments
- **Updates**: Share progress on projects they've supported
- **Annual summaries**: End-of-year giving statements for donors

### Segmentation

Organize donors for targeted communication:

- By giving level (major donors, recurring, one-time)
- By campaign or project supported
- By recency and frequency
- Custom tags and categories

### Export Capabilities

Your data is yours:

- Export donor lists in standard formats
- Integrate with your existing CRM
- Download transaction history
- Access API for custom integrations

→ [API overview](../api/overview.md)

## Dashboard

### Real-Time Overview

See your organization's activity at a glance:

- Donations today, this week, this month
- Active campaigns and progress
- Upcoming payouts
- Key metrics and trends

### Analytics

Understand your fundraising performance:

- Donor acquisition and retention
- Campaign performance comparison
- Average gift size and trends
- Recurring giving growth
- Source attribution (where donors come from)

### Notifications

Stay informed:

- New donation alerts
- Large gift notifications
- Payout confirmations
- Verification updates

## Transparency Features

### Public Profile

Your organization's public face on Amply:

- Organization information and mission
- SDG alignment and focus areas
- Transparency metrics
- Verified status badge
- Public transaction ledger

### Transparency Dashboard

Real-time transparency metrics:

| Metric | Description |
|--------|-------------|
| Visibility score | Percentage of funds fully public vs. aggregated |
| Update frequency | How often financial data is published |
| Verification level | Depth of organizational verification |
| Response rate | Speed responding to donor inquiries |

### Privacy Controls

Manage what's visible:

- Set visibility levels for spending categories
- Aggregate sensitive transactions (HR, legal, security)
- Control donor information visibility
- Manage team access levels

→ [Transparency & Privacy](../transparency/privacy.md)

## Fund Management

### Funds and Projects

Organize donations by purpose:

- **General fund**: Unrestricted donations
- **Project funds**: Donations for specific initiatives
- **Restricted funds**: Legally restricted purposes
- **Emergency funds**: Rapid-response giving

### Fund Tracking

Follow money from donation to impact:

- Allocate incoming donations
- Track spending by fund
- Report on fund utilization
- Transfer between funds (with documentation)

| Role | Capabilities |

| Admin | All settings, user management |
| Finance | Transactions, payouts, financial reports |
| Content | Profile, campaigns, communications |

- Who did what, when

- Access logs available
- Exportable for compliance
