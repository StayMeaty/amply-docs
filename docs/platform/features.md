# Platform Features

Explore the features that make Amply a powerful platform for charitable giving.

## Donor Features

### Personal Dashboard

- View all donations in one place
- Track giving history and patterns
- Monitor impact across organizations
- Manage payment methods

### Smart Giving

- Set up recurring donations
- Create giving budgets
- Schedule future donations
- Round-up spare change

### Discovery

- Browse by category or cause
- Search for specific organizations
- View featured campaigns
- Get personalized recommendations

### Social Features

- Share your impact
- Create giving circles
- Challenge friends to give
- Celebrate milestones

## Organization Features

### Profile Management

- Customizable organization page
- Media gallery for photos and videos
- Story and mission showcase
- Team member profiles

### Campaign Tools

- Create targeted fundraising campaigns
- Set goals and track progress
- Time-limited appeals
- Matching gift campaigns

### Donor Management

- View supporter analytics
- Send thank-you messages
- Segment donor communications
- Export donor data

### Reporting Dashboard

- Real-time donation tracking
- Impact metric submission
- Automated report generation
- Financial summaries

## Technical Features

### Security

- End-to-end encryption
- PCI-compliant payments
- Two-factor authentication
- Regular security audits

### Integration

- API access for developers
- Webhook notifications
- CRM integrations
- Accounting software compatibility

### Accessibility

- WCAG 2.1 compliant
- Screen reader support
- Keyboard navigation
- High contrast mode
